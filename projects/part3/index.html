CSCE242 CubeSat Documentation Project

A CubeSat is a small, short time use satellite that is used for small missions that larger, more expensive satellitescannot complete. This webpage will hold the documentation of the project such that future classes can access the information for when they design the CubeSat V3. Documentation should consist of each stage of the project. This being the initial project planning phase, the conceptual design phase, the detailed design phase, construction phase, and the finalization phase. Each section will include documents created and used as well as descriptions of the thought processes.
