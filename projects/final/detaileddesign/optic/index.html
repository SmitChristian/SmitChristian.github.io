The entire purpose of the CubeSat is to capture images of Earth for transmission back to the surface. To do this a camera had to be decided upon. After browsing the opens available off the shelf, it was decided to use the RaspberryPiCam3 as it has the most troubleshooting support as well as the most direct compatibility with the Pi4. The version of the PiCam that we selected was the 12MP 120-degree wide angle lens. Choosing the wide-angle lens allows for us to see a lot more of earth in each image that is taken. The focus of our images will be 1,920,000 km2 from an orbit of 400km. While the default resolution of the camera is 12 mega pixels, the minimum design of image quality is 480x480p which is 230 kilo pixels

This will help reduce image file size making it possible to transmit the images in a time efficient manner. The file size of the images we have captured range from 5-10kb being well within the budgeted size to allow for complete transmission before the next image is captured. This extra time allotment allows for us to implement forward error correction to help ensure that the images are received in whole.
