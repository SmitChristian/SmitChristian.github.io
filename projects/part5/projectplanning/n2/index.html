N2 Chart

The N2 diagram outlines the different systems that the CubeSat will have and shows how they interact with one another. Functions above the curve are interactions from the system to the left to the system below and functions below the curve are interactions from the system to the right to the system above. The processing is the only system to have two-way interactions with every other system aboard the CubeSat and can be thought of as the central system of the CubeSat. All the functions pertaining to the ground station(s) are closely linked to data transmission so image processing, display, and the independent power and processing within the ground station fall within data transmission.
