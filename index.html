Christian Smith's CSCE242:Web Development Class Page

In this class, students are tasked to learn to use HTML and CSS to design the front-end of a custom website of their making. Towards the end of the course, students will learn to use node and server side JavaScript to store and retrieve server side data. Overall, the students should come out of the course with some level of understanding of HTML, CSS, and Javascript.

Assignments

Assignment 1 - Basic HTML

This assignment covers the Europa Clipper space mission. It was created to gain an understanding of base level of HTML. It was updated after the completion of Assignment 2.

Assignment 2 - Basic CSS

This assignment covers the Psyche space mission. It was created to gain an understanding of basis CSS and style sheets.

Assignment 3 - Page Layout

This assignment covers the Swarm space mission. It was created to gain an understanding and establish page layouts.

Assignment 7 - Introduction to Javascript

This assignment covers different ways to die in space. It was created to gain an understanding of JavaScript.

Projects

Part 1 - Project Definition

This project part covers the topic proposal for the project. The contained pdf outlines the idea for the project and the expected use for it

Part 2 - Project Wireframe

This project part covers the wireframe of the project website page. This is used to gain a better understanding of what the page's layout will look like in the end.

Part 3 - HTML and CSS

This project part covers the intial HTML and CSS setup of the project website page.
